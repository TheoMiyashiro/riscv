Simplified RISC-V

This project is a simplified RISC-V processor developed for the Computer Architecture course

It has:

Hard-coded instructions

5-stage pipeline (IF, ID, EX, MEM, WB)

Basic hazard handling

Supported Instructions

lw, sw, add , sub, and, or, beq

Purpose

The goal is to understand how a pipelined RISC-V CPU works and how hazards are handled in a simple implementation.
